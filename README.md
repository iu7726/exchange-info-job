## Description
Exchange info Snapshot Project<br/>

## Features
- BillsJob: Collects filled orders.
- OkxBalaanceJob: Collect the balance of registered users.
- RebateJob: Records the broker's profits.

## Install

```
$ yarn add
```

## Running
```
$ yarn watch
$ yarn start
```
